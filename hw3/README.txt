mushrooms: contains scripts that use the Mushrooms dataset
pima: contains scripts that use the Pima Indians diabetes dataset
paper: contains the tex, pdf and other files for the paper

Inside folder pima:

clustering1: contains initial clustering script (clustering.py)
clustering2: contains script for clustering after PCA is performed (clustering.py)
clustering3: contains script for clustering then running a neural network on the data (clustering.py)
pca: contains script to reduce the dimensions (pca.py)
nn_pca: contains script to reduce the dimensions and then run a neural network (nn_pca.py)

Inside folder mushrooms:

clustering1: contains initial clustering script (clustering.py)
clustering2: contains script for clustering after PCA is performed (clustering.py)
pca: contains script to reduce the dimensions (pca.py)
